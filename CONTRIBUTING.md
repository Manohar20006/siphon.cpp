# Contributing

This project is a llama.cpp fork focused on MoE expert streaming and CUDA
inference experiments.

Please keep changes scoped and reproducible:

- include the build command used
- include the model/configuration used for performance claims
- keep generated traces, logs, profiles, and GGUF files out of git
- put long-form notes in `docs/`
- put reusable expert lists in `expert-suites/`

For the original upstream llama.cpp contribution guide, see:

```text
docs/CONTRIBUTING.md
```
